# Docker-curl
---
Like it says, it's a docker image built on alpine with curl installed and only 7 MB!!

## Usage

```console
$ docker run --rm jibsoft/curl curl -fsSL https://www.google.com/
```

## Tags

* `jibsoft/curl:latest` based on `alpine:latest`
